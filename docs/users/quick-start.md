# Quick Start

This guide gets you writing in minutes.

## Option A: Use TON3S Online (No Install)

1. Open [https://ton3s.com](https://ton3s.com).
2. Start typing.
3. Your note is auto-saved locally in your browser.

## Option B: Self-Host with Docker

```bash
git clone https://github.com/hypercoind/TON3S.git
cd TON3S
docker compose up -d
```

Open `http://localhost:3002`.

Stop the app:

```bash
docker compose down
```

## First Session Checklist (Recommended)

1. Create your first note.
2. Change theme and font.
3. Export a JSON backup.
4. Optionally connect a Nostr signer and publish.
